# Ayo The Pizza Here
## Pentagon Pizza Tracker Discord Bot

# How does it work?
It uses PizzInt (https://pizzint.watch) to get the current "doughcon" level and out put it with an emoji/image.
You can, if you wish, customise the icons it uses for the thumbnails in the config.py you make.

# Setup
Run ```pip install -r requirements.txt``` to install requirements, then duplicate 
```config.py.template``` and remove the .template extention. Then, add your bot token
in the ```token = ""``` area. You can make the bot at https://discord.com/developers/applications.
After that, run bot.py
